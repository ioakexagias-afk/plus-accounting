# Plus Accounting — Document Generator

Εφαρμογή δημιουργίας Προσφορών και Ιδιωτικών Συμφωνητικών.

## Ανέβασμα στο Vercel (δωρεάν, 5 λεπτά)

### Βήμα 1 — GitHub
1. Πήγαινε στο https://github.com και δημιούργησε λογαριασμό (αν δεν έχεις)
2. Πάτα **"New repository"** → δώσε όνομα π.χ. `plus-accounting`
3. Πάτα **"Create repository"**
4. Ανέβασε τα αρχεία: πάτα **"uploading an existing file"** και σύρε τον φάκελο `plusaccounting`

### Βήμα 2 — Vercel
1. Πήγαινε στο https://vercel.com και κάνε **"Sign Up with GitHub"**
2. Πάτα **"Add New Project"**
3. Επίλεξε το repository `plus-accounting`
4. Πάτα **"Deploy"** — τελειώνει σε ~2 λεπτά
5. Παίρνεις URL τύπου: `https://plus-accounting-xxx.vercel.app`

Αυτό το URL ανοίγει από οπουδήποτε — κινητό, tablet, άλλος υπολογιστής.
